#Movie Trailer Website
This server-side code to store a list of my favorite movies, including box art imagery and a movie trailer URL. It serves this data as a web page allowing visitors to review their movies and watch the trailers.

#Quickstart
##Install
*Install [Python](https://www.python.org/downloads/)
*Run _Entertainment.py_ in the files
